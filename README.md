# 📁 Certificate Vault

A secure and efficient **Certificate Management System** built using Python. This project allows users to store, manage, and access digital certificates in an organized way.

---

## 🚀 Features

* 📤 Upload and store certificates
* 📂 Organized file management system
* 🔍 Easy access to stored certificates
* 💾 Automatic database handling (SQLite)
* 🧩 Simple and user-friendly structure

---

## 🛠️ Technologies Used

* **Python**
* **SQLite**
* **File Handling**
* **OS Module**

---

## 📂 Project Structure

```
CertificateVault/
│
├── main.py
├── database.db   (auto-created)
├── certificates/
└── README.md
```

---

## ⚙️ How to Run the Project

1. Clone the repository
2. Navigate to the project folder
3. Run the main file

```
python main.py
```

---

## 📌 Notes

* The `database.db` file will be created automatically when the project runs
* The `certificates/` folder is used to store uploaded files
* Ensure the folder exists before running the project
